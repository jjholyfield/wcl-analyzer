What Needs to Change
1. P2 Phase Wall: Cosmic Barrier is wiping the raid

Cosmic Barrier killed 60 players in P2 alone — more than any other single mechanic in any phase. This is the Rift Simulacrum's shield that needs to be broken before it expires. When it fails, it wipes the raid with massive AoE. DPS need to swap to the shield immediately when it spawns and burn it before the cast completes.

2. P2: Simulacrum Backlash + Bursting Emptiness (18 + 16 deaths)

Simulacrum Backlash killed 18 players in P2. This fires when the Rift Simulacrum dies or when Cosmic Barrier breaks — the raid needs to be spread and healthy when this happens. Bursting Emptiness added 16 more deaths from Void Droplet soaks. Healers should have CDs planned for Cosmic Barrier breaks.

3. P1: Clean up early deaths (15/20 pulls had first death in P1)

Silverstrike Arrow (16 deaths), Void Remnants (15 deaths), and Bursting Emptiness (13 deaths) are the main P1 killers. 15 of 20 pulls had their first death in P1. Pulls with clean P1 execution lasted significantly longer.

4. P3 execution needs reps

The raid only reached P3 in 4/20 pulls (20%). Voidstalker Sting (10 deaths) and Cosmic Barrier (13 deaths) are the main killers in P3. The death count per P3 pull is inflated by wipe-cascade deaths at end of pull. More P3 time will come naturally as P2 execution improves — the focus should be breaking the P2 wall first.

5. What's working

Night 2 best was barely scratching the boss. Night 3 best is 13.55% — deep P3 territory. That is a massive jump. The late session showed consistent P3 entries and pulls over 5 minutes. Raid DPS is strong when people are alive — the issue is survival, not throughput. The Intermission is largely clean (32 deaths total, only 2.3 per pull). P1 execution improved steadily through the night. The team is learning this fight.
